Regras de Nomenclatura

  • OSs: [SIGLA]-OS XXX-AAAA - [Descrição]
  • Numeração com 3 dígitos: 001, 002, 003
  • NFe: NFe NNN - YYYYMMDD - [Contrato] - R$ X.XXX,XX.pdf
  • TRD: TRD OS XXX-AAAA - [Descrição]
  • Certidão: Certidão do GDF - YYYYMMDD.pdf
  • SICAF: SICAF YYYYMMDD.pdf
  • Ofício: Ofício SEA-OF-AAAA-NNN-[CLIENTE]

Configurar acesso ao GitHub

Para salvar alterações diretamente no repositório, você precisa de um Personal Access Token (PAT) do GitHub.

Como criar:
1. Acesse GitHub → Settings → Tokens
2. Clique em "Generate new token" (Fine-grained)
3. Nome: Painel Drives
4. Em "Repository access" selecione "Only select repositories" → qualidade
5. Em "Permissions" → "Contents" → Read and write
6. Clique em "Generate token" e copie o token

O token fica salvo apenas no seu navegador (localStorage). Ninguém mais tem acesso.
Token salvo! Agora clique em "Salvar no GitHub" novamente.
